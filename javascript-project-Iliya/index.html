In simple words, a regular expression is a set of characters that sets certain rules for searching and matching text. You can use regular expressions to find certain strings in the text, replace parts of the string with other values, or extract the necessary information from the text.

json

In simple words, JSON is a way of presenting data as text so that it can be easily understood and used by various programs. It uses simple rules to describe objects and arrays, as well as to represent simple data types such as strings, numbers, boolean values, etc.

event loop

In simple words, an event loop is like an attendant who continuously checks whether an event has occurred and reacts to it when it occurs. It has a list of tasks that need to be completed, and it iterates through this list of events one by one, processing each event when it occurs.
