Delegating events

In JavaScript, event delegation is when we use a single event handler for a group of elements, instead of assigning a separate handler to each element. When an event occurs on one of the elements, it is intercepted by a common handler, which determines which element triggered the event and performs the appropriate actions.

Class List

classList in JavaScript is a property that allows us to work with HTML element classes. With the help of classList, we can add, delete, or check for the presence of classes in an element. Let's say we have an element on a web page and we want to change its classes using JavaScript. Instead of accessing the class attribute directly, we can use the classList property of this elem

add()

The add method allows you to add a new class to the list of element classes. For example, element.classList.add('new-class') will add a class named "new-class" to the element.

remove()

The remove method allows you to remove a class from the list of element classes. For example, element.classList.remove('old-class') will remove a class named "old-class" from the element.

contains()

The contains method allows you to check whether an element contains a certain class. It returns true if the class is present, and false if it is not. For example, element.classList.contains('highlight') will return true if the element has a "highlight" class.
